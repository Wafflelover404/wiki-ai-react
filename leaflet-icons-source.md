## How to Download for Canva

### Method 1: Direct SVG Download
1. Go to any icon URL (e.g., https://lucide.dev/icons/lightbulb)
2. Click the "Download SVG" button
3. Save the SVG file to your computer
4. Upload to Canva

### Method 2: Copy SVG Code
1. Go to any icon URL
2. Click "Copy SVG" button
3. Save the code to a text file with .svg extension
4. Upload to Canva

### Method 3: Bulk Download
1. Visit https://lucide.dev/icons/
2. Select multiple icons
3. Download as a ZIP file
4. Extract and upload to Canva

## Canva Tips
1. **Upload SVGs**: Go to "Uploads" → "Upload files" → Select SVG files
2. **Change Colors**: Select icon → Click on color palette → Choose your brand colors
3. **Resize**: Drag corners to resize while maintaining proportions
4. **Group Elements**: Select multiple icons → Right-click → Group for easier alignment

## Recommended Sizes for Print
- **Header Icons**: 24-32px
- **Feature Icons**: 16-20px  
- **Placeholder Icons**: 48-64px
- **Contact Icons**: 20-24px
